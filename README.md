# IP VAULT

A high-security, encrypted digital vault for sensitive assets. Built with Electron, Tailwind CSS, and AES-256-GCM encryption.

## Features
- **Military Grade Encryption**: Uses AES-256-GCM with Scrypt key derivation.
- **Biometric Aesthetic**: Premium "Aurora Glass" UI with smooth transitions.
- **Asset Categories**: Separate buckets for Visuals, Media, Documents, and Encrypted Drops.
- **Zero Knowledge Architecture**: All encryption happens locally; master key is never stored.

## Tech Stack
- Electron
- Lucide React (Icons)
- Anime.js & GSAP (Animations)
- Crypto-js & Node Crypto

## Setup
```bash
npm install
npm start
```
