# 🛡️ IP VAULT — Stark Intelligence OS Edition

A high-fidelity, secure asset encryption vault built for the next generation of digital privacy. Inspired by the Stark Industries HUD, this application provides an immersive security experience.

## ✨ Advanced Features

- **AES-256-GCM Encryption**: Industrial-grade security for all visual and document assets.
- **Biometric Hub**: Simulated identity verification screen with high-tech scan animations.
- **Secure Intel Notes**: Built-in encrypted text editor for storing private codes and messages.
- **Satellite Radar Tracker**: Real-time asset "tracking" simulation (Radar UI).
- **Persistent Metadata**: All records are encrypted and stored locally on disk for zero data loss.
- **Intelligence Operator Log**: Live system feed tracking user actions and sessions.
- **Inactivity Armor**: Automatic session termination after 2 minutes of idle time.
- **Drag & Drop Uplink**: Encrypt files instantly by dropping them onto the Command Centre.
- **Visual Archives**: Native image previews for encrypted assets in the vault.

## 🚀 Getting Started

1. **Clone & Install**:
   ```bash
   git clone [Repo-URL]
   cd ip-vault
   npm install
   ```

2. **Launch System**:
   ```bash
   npm start
   ```

## 🛠️ Tech Stack
- **Electron**: Desktop Framework
- **Tailwind CSS**: High-Tech UI Layout
- **Node.js Crypto**: AES-GCM Encryption
- **Fs-Extra**: Secure File Management
